DMN Tabellen können vielfältige und umfangreiche Informationen enthalten. Aus diesem Grund bietet das Tool eine Suchen und Ersetzen-Funktion um Daten aufzufinden und Massenweise anzupassen.

Suche öffnen

Die Suche kann über das Bearbeiten Menü oder mit dem Hotkey STRG-F geöffnet werden. Die Suche durchsucht ausschließlich geöffnete DMN Dateien und dockt sich direkt innerhalb des Editor-Bereichs ein.

Die Suchfunktionen

Suchwort

Zeichenkette die gesucht werden soll. Die Suche sucht in jedem Datenfeld der Tabelle ob die Zeichenkette enthalten ist. Wird die Suche innerhalb der DRD Ansicht (Tabellenübersicht) geöffnet werden alle Tabellen der Datei durchsucht.

Alle Zeilen die das gesuchte Wort nicht enthalten werden nach der Suche ausgeblendet. In der DRD Ansicht werden alle Tabellen ausgeblendet die keine gefundenen Zeilen enthalten.

Suchspalte

Mit Hilfe einer gewählten Suchspalte kann die Suche auf eine konkrete Spalte Input oder Output beschränkt werden. Die Annotation-Spalte kann ebenfalls durchsucht werden.

Minimieren

Der Suchdialog kann minimiert werden. Wird der Suchdialog minimiert, kann in der DMN Datei navigiert und zwischen den Tabellen gewechselt werden ohne das die Suchparameter verloren gehen.

Ersetzen

Die Ersetzen Funktion wird über den Pfeilknopf auf der linken Seite geöffnet. Der Suchdialog erweitert sich hierbei nach unten um die Eingabefelder zum Ersetzen von Daten.

Alle Ersetzungen werden stets auf die sichtbaren Zeilen angwendet. Die Suche schränkt somit die Zeilenzahl ein in der Daten ersetzt werden. Innerhalb der gesuchten Zeilen kann dennoch ein Suchbegriff den exakten zu ersetzenden Wert einschränken.

Nach der Durchführung einer Ersetzung wird die Anzahl der Ersetzten Begriffe angezeigt.

Ersetzen-Funktionen

Ersetzen

Definiert die Zeichenkette die ersetzt werden soll. Kann leer gelassen werden. Ein leerer Wert füjhrt dazu das der Ersatzwert in alle leeren Zellen eingefügt wird.

Ersetzen mit

Definiert den Bergiff mit dem die definierte Zeichenkette ersetzt wird. Ist die Suchzeichenkette leer, wird dieser Wert in alle leeren Zellen eingesetzt.

In Spalte Schränkt die Ersetzen Funktion auf genau eine definierte Spalte ein.

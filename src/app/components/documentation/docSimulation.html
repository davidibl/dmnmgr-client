Die Simulation kann genutzt werden um Entscheidungen oder Entscheidungsbäume zu testen. Um eine Entscheidung zu testen muss stets ein Eingabedatenmodell definiert sein.

Um eine Simulation durchzuführen muss ein Simulationsservice erreichbar und konfiguriert sein. Der Simulationsservice ist ein REST Webservice, über den eine DMN basierte Entscheidung ausgeführt werden kann.

Einstellungen

Das Einstellungsmenü kann über Bearbeiten -> Einstellungen (STRG + K) erreicht werden. Unter Simulator Basis Url muss die gültige URL eines Simulations Service angegeben sein.

Simulation

Das Simulationsmenü findet sich in der Ansicht Dein Modell.

Simulationsmenü öffnen

Wurde ein Datenmodell definiert (siehe Der Data Modeler), können nun in diesem Schema in einer Baumansicht Eingabedaten angegeben werden, auf deren Basis die Entscheidungstabelle ausgeführt wird.

Enthält die DMN Datei Fehler, wird der Fehler unterhalb der Eingabedaten angezeigt. Treffer werden in der Tabelle grün hinterlegt.

Wird keine zu den Eingabedaten korrespondierende Regel gefunden, wird eine entsprechende Nachricht ausgegeben.

Alle gefundenen Regeldatensätze werden unterhalb der Eingabedaten angezeigt. Alle Regeln die den Eingabedaten entsprechen, werden grün hinterlegt.

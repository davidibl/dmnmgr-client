Mit dem DMN Manager können DMN Dateien bearbeitet und Git Repositories verwaltet werden. DMN kann der umfangreichen Konfiguration von Anwendungen sowie der visuellen Definition von Geschäftsregeln in einer Standardisierten Sprache dienen. Dieses Tool soll hierbei die Zusammenarbeit an Software fördern, indem es einen einfachen Zugang zu GIT und einer Anwendungsnahen Definitionssprache von Geschäftsregeln bietet.

Get Started

Der DMN Manager arbeitet mit dmnapp.json Dateien. Bestehende DMN Dateien können nur in ein DMN Manager Projekt importiert werden.

  • Neues Projekt
    Nach dem Start des Tools kannn direkt modelliert werden. Einfach in den Dein Modell Tab wechseln und Regeln definieren. Sobald gespeichert wird, entsteht die DMN Manager Projektdatei automatisch.
  • bestehende DMN Datei
    Um mit einer bestehenden DMN Datei zu arbeiten muss diese zunächst in ein DMN Manager Projekt importiert werdem. Hierzu einfach Datei -> DMN Datei importieren wählen. Wird danach gespeichert wird die DMN Manager Projekt Datei direkt mit abgelegt.
